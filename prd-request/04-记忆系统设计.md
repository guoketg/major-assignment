## 1. 三层记忆架构

```
┌─────────────────────────────────────────────────────────────┐
│                    长期记忆 (Long-term)                       │
│  跨会话持久化 · 向量检索 · Chroma/FAISS                     │
│  内容：论文特征向量、用户偏好、历史实验知识                  │
├─────────────────────────────────────────────────────────────┤
│                    工作记忆 (Working)                        │
│  当前会话内持久 · JSON 文件 · 检查点 checkpoint              │
│  内容：papers_archive, innovation_candidates, experiment_log  │
├─────────────────────────────────────────────────────────────┤
│                    短期记忆 (Short-term)                     │
│  内存中 · AgentState · 节点间流转                           │
│  内容：messages, current_agent, sub_task_queue               │
└─────────────────────────────────────────────────────────────┘
```

## 2. 短期记忆 — AgentState

LangGraph 的 `StateGraph` 内置短期记忆，状态在节点间自动传递。

```python
class AgentState(TypedDict):
    messages: List[Dict]          # 对话历史（每次追加新消息）
    current_agent: str            # 当前路由目标
    sub_task_queue: List[Dict]    # 待执行子任务
    memory: Dict                  # 工作记忆的当前快照
    # ... 其他字段
```

**特性：**
- 每个节点读取并修改 state
- 修改后的 state 自动传给下一个节点
- 支持 `MemorySaver` 检查点持久化

### 记忆更新时机

| 操作 | 触发节点 | 更新内容 |
|------|----------|----------|
| 检索论文 | Research Agent | `papers_archive.append(...)` |
| 构思创新 | Innovator Agent | `innovation_candidates = [...]` |
| 实验反馈 | Experiment Agent | `experiment_log.append(...)` |
| 切换话题 | Supervisor | `research_topic`, `current_focus` 重置 |

## 4. 记忆检查点（Checkpoint）

使用 LangGraph 的 `MemorySaver` 实现会话内持久化：

```python
from langgraph.checkpoint import MemorySaver

checkpointer = MemorySaver()
app = workflow.compile(checkpointer=checkpointer)

# 每次节点执行后自动保存检查点
# 下次执行时可以从检查点恢复
```

对于跨会话持久化，将工作记忆序列化为 JSON 文件：

```python
import json
import os

MEMORY_DIR = "agent_memory"

def save_working_memory(session_id: str, memory: dict):
    path = os.path.join(MEMORY_DIR, f"{session_id}.json")
    with open(path, "w") as f:
        json.dump(memory, f, ensure_ascii=False, indent=2)

def load_working_memory(session_id: str) -> dict:
    path = os.path.join(MEMORY_DIR, f"{session_id}.json")
    if os.path.exists(path):
        with open(path) as f:
            return json.load(f)
    return {}
```

## 6. 记忆在 Agent 感知中的应用

每个 Agent 在执行前从记忆中检索相关上下文：

```python
def retrieve_for_agent(agent_name: str, state: AgentState) -> dict:
    """为指定 Agent 提取相关记忆片段"""
    context = {}
    
    if agent_name == "research_agent":
        # 提供已有论文索引，避免重复检索
        context["existing_papers"] = state["memory"].get("papers_archive", [])
        context["research_topic"] = state["memory"].get("research_topic", "")
        
    elif agent_name == "innovator_agent":
        # 提供调研结果
        context["papers_archive"] = state["memory"].get("papers_archive", [])
        context["baseline"] = state["memory"].get("baseline_model", "")
        
    elif agent_name == "experiment_agent":
        # 提供实验历史和当前创新方案
        context["experiment_log"] = state["memory"].get("experiment_log", [])
        context["innovation_candidates"] = state["memory"].get("innovation_candidates", [])
    
    return context
```
