## 1. 项目定位

面向深度学习研究者的**协同科研分析系统**。核心价值不是简单的对话或搜索，而是通过多个分工明确的 AI Agent 协作，辅助研究者完成 **「调研 → 创新构思 → 实验设计 → 反馈迭代 → 报告生成」** 的完整科研闭环。

## 2. 核心能力

| 能力 | 说明 |
|------|------|
| **文献调研** | 多维度检索 arXiv，按「已证实/争议/不足」分类整理 |
| **创新构思** | 针对同一问题提出多种创新路径，多维度对比 |
| **实验设计** | 将创新拆解为可执行的消融实验方案 |
| **迭代优化** | 接收用户实验反馈，定位瓶颈，调整方案 |
| **报告导出** | 自动生成 Word 调研报告、Draw.io 架构图 |

## 3. 多 Agent 协作模式

```
用户输入
    │
    ▼
┌─────────────────────────────────────────────────────────┐
│                   Supervisor Agent                       │
│  意图识别 → 任务分解 → 路由分发 → 结果综合               │
└────────┬────────┬────────┬────────┬──────────────────────┘
         │        │        │        │
    ┌────┘  ┌─────┘  ┌────┘  ┌────┘
    ▼       ▼        ▼       ▼
┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐
│对话   │ │文献   │ │创新   │ │实验   │
│Agent  │ │Agent  │ │Agent  │ │Agent  │
└──────┘ └──────┘ └──────┘ └──────┘
    │       │        │       │
    └───────┴────────┴───────┘
                │
                ▼
┌──────────────────────────────────┐
│        综合响应合成               │
│  Supervisor 整合各 Agent 输出     │
└──────────────────────────────────┘
```

## 4. 与现有系统的关系

当前项目已实现：
- ✅ FastAPI 后端 + React 前端
- ✅ 流式对话（SSE）
- ✅ 多模型选择
- ✅ arXiv 搜索接口
- ✅ 会话历史管理

多 Agent 系统将**复用这些基础设施**，把 `/chat/stream` 端点改造为 LangGraph 的入口，arXiv 搜索降级为 Research Agent 的工具之一，而非独立的 UI 面板。

## 5. 技术栈

| 层 | 技术 |
|----|------|
| Agent 框架 | LangGraph (StateGraph) |
| LLM | DeepSeek Chat / Reasoner（通过 OpenAI SDK） |
| 后端 | FastAPI（复用现有） |
| 前端 | React（复用现有，增加 Agent 思考过程展示） |
| 工具 | arXiv API, python-docx, drawio |
| 记忆 | 短期（State）+ 工作（JSON/文件）+ 长期（向量库） |
| 部署 | Docker Compose（不变） |
