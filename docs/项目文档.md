## 功能说明

### 1. 多轮对话

支持连续对话，历史记录自动保存。

**模型选择**（前端下拉框即时切换）：

| UI 选项 | 对应 API 模型 | 特点 |
|---------|--------------|------|
| DeepSeek V4 Flash | `deepseek-chat` | 快速响应，默认 |
| V4 Pro | `deepseek-chat` | 更强能力 |
| 思考模式 | `deepseek-reasoner` | 深度推理，显示思考过程 |

> 切换模型只需在前端下拉框中选择，**无需重启容器**。每次发送消息时都会携带当前选中的模型。

**思考过程展示**：
- 模型思考时显示 `🤔 思考中...`（展开状态）
- 模型输出正式内容后自动折叠为 `🧠 思考过程`
- 用户可随时点击展开/折叠查看完整思考内容

### 2. arXiv 论文搜索

点击顶部的 **📄 论文搜索** 标签切换到搜索面板。

**搜索语法**（与 arXiv 官网一致）：

| 前缀 | 说明 | 示例 |
|------|------|------|
| `all:` | 全文搜索 | `all:machine+learning` |
| `ti:` | 标题 | `ti:transformer` |
| `au:` | 作者 | `au:"Bengio, Y"` |
| `cat:` | 分类 | `cat:cs.LG` |
| AND/OR/NOT | 布尔逻辑 | `cat:cs.AI+AND+ti:attention` |

**搜索结果** 显示：标题（可点击链接）、作者、分类、日期、摘要。

### 3. 流式输出（SSE）

- 后端使用 Server-Sent Events (SSE) 逐 token 推送
- 前端逐字渲染，带闪烁光标 `|`
- 流式过程中用户可自由滚动查看历史消息，不会被自动拉回底部

### 4. 窗口功能

- 左侧：会话历史列表（可新建/删除会话）
- 右侧顶部：标签切换（对话/论文搜索） + 模型选择
- 右侧主区域：消息/搜索结果展示
- 底部：输入框 + 发送按钮

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/chat/stream` | 流式对话（SSE） |
| POST | `/chat` | 非流式对话 |
| GET | `/sessions` | 会话列表 |
| POST | `/sessions` | 创建新会话 |
| GET | `/history/{session_id}` | 会话历史 |
| DELETE | `/history/{session_id}` | 删除会话 |
| GET | `/arxiv/search` | arXiv 论文搜索 |
| GET | `/health` | 健康检查 |

## 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `DEEPSEEK_API_KEY` | — | DeepSeek API 密钥（必填） |
| `DEEPSEEK_MODEL` | `deepseek-chat` | 默认模型（前端切换会覆盖） |
| `OPENAI_BASE_URL` | `https://api.deepseek.com/v1` | API 地址 |
| `REACT_APP_API_URL` | `http://localhost:8000` | 前端调用的后端地址 |
